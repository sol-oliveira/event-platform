 ### 🖥️ Rodando a Aplicação 

```bash
# Clonar repositório
$ git clone https://github.com/sol-oliveira/ignite-lab.git

# Instale as dependências
$ npm install

# Execute a aplicação em modo de desenvolvimento
$ npm run dev
```
